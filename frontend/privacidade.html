Política de Privacidade

Última atualização: 30 de junho de 2026

Resumo: Coletamos apenas o número de WhatsApp para comunicação sobre o serviço. Não armazenamos IPs, não vendemos dados e respeitamos integralmente a LGPD.

1. Quem Somos

Este site é operado por Luciano Júnior, editor de vídeo profissional. Para contato: +55 75 8138-5296.

2. Dados Coletados

Coletamos apenas os dados estritamente necessários para a prestação do serviço:

  • Número de WhatsApp: fornecido voluntariamente pelo usuário ao clicar nos botões de contato.
  • Nome (opcional): informado pelo usuário durante a conversa no WhatsApp.

Não coletamos: endereços IP, cookies de rastreamento, dados de navegação, localização ou qualquer outra informação pessoal sensível.

3. Banco de Dados (Cloudflare D1)

O banco de dados deste site (Cloudflare D1) armazena apenas as configurações do painel administrativo (preço, número de WhatsApp e textos do site). Nenhum dado de visitante é armazenado no banco de dados.

4. Finalidade do Uso dos Dados

O número de WhatsApp é utilizado exclusivamente para:

  • Comunicação sobre o serviço de edição de vídeo contratado.
  • Envio do arquivo editado e acompanhamento do projeto.
  • Suporte pós-entrega e revisões.

5. Compartilhamento de Dados

Os dados pessoais não são vendidos, alugados ou compartilhados com terceiros, exceto quando exigido por lei ou ordem judicial.

6. Segurança

As comunicações são realizadas via WhatsApp (criptografia de ponta a ponta). O painel administrativo é protegido por chave de acesso. A infraestrutura do site utiliza Cloudflare, com SSL/TLS obrigatório e proteção DDoS.

7. Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados.
  • Acessar os dados que possuímos sobre você.
  • Solicitar a correção de dados incompletos ou incorretos.
  • Solicitar a exclusão dos seus dados.
  • Revogar o consentimento a qualquer momento.

Para exercer qualquer um desses direitos, entre em contato via WhatsApp: +55 75 8138-5296.

8. Retenção de Dados

Os dados de contato são mantidos apenas pelo período necessário para a prestação do serviço e por até 90 dias após a conclusão do projeto, para fins de suporte.

9. Cookies

Este site não utiliza cookies de rastreamento, analytics ou publicidade. Não há nenhum script de terceiros que colete dados dos visitantes.

10. Alterações nesta Política

Eventuais alterações serão publicadas nesta página com a data de atualização. O uso continuado do site após as alterações implica aceitação da nova política.

11. Contato

Dúvidas sobre privacidade? Fale conosco: +55 75 8138-5296.
